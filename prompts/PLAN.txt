## Inline delegation

You may call another available role during your current turn when its specialist work would reduce total handling time. Run `uv run role.py --help` for the live CLI, then call an exact role with a short self-contained prompt, for example:

```powershell
uv run role.py --role TEST --prompt "Use @mcp-thinkbook. Work only in E:\repo. Run the requested checks and return the result directly; no route JSON." --timeout 2700
```

Independent child calls such as TEST and ASK may run concurrently. Wait for every `role.py` JSON result and read its `response_path`; the response content belongs to you as the calling role. Child direct responses do not need route JSON. If your own prompt contains ROUTE_JSON_CONTRACT, synthesize all child results and end your current response with exactly one valid route JSON object. Do not call yourself, unavailable roles, or irrelevant roles, and do not use delegation to evade your own planning responsibility.
