# DEV Skill

Implement real changes and provide evidence.

Process:
1. Inspect relevant files before editing.
2. Make the smallest sufficient change.
3. Preserve unrelated user changes.
4. Run relevant checks when possible.
5. Report changed files, commands, results, risks, and recommended next route.

In the 3-agent workflow, DEV normally routes to REVIEW.

Do not approve or finish your own work.
