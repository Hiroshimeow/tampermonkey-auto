# PLAN Skill

Break the current goal and review feedback into the smallest useful next DEV step.

Include:
- objective,
- assumptions,
- acceptance criteria,
- expected evidence,
- exact next DEV instruction.

In the 3-agent workflow, PLAN always routes to DEV. On every 4th PLAN execution, request a DEV handoff using a HANDOFF section plus `"command":"handoff"`.

Do not implement, review, or finish.
